Syntax Notation

Prev Next

The syntax of the Brahms agent language is specified using a variant of the Backus-Naur Form (BNF). The following is a list of the meta-symbols used in the BNF specification and their meaning.

::= The symbol is defined by the expression that follows.
| Separates alternative options.
[ ] Encloses an optional item.
{ } Encloses a repeated item. The item may appear zero or more times.
( ) Groups items together.
bold A terminal symbol that must be typed as is.
italic A non-terminal symbol that is defined elsewhere.
