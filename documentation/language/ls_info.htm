This document specifies the Brahms agent language. This document is intended for Brahms users who want to develop Brahms models. This document is not intended as a tutorial. A separate tutorial document is available for Brahms users who want to learn how to develop Brahms models. This document is a reference document specifying the syntax and semantics of the Brahms agent language.

This document is organized as follows. Chapter 2 specifies the syntax notation used in this document. Chapter 3 gives an overview of the concepts in a Brahms model. Chapters 4 through 14 specify the Brahms agent language. Chapter 15 specifies how to develop Java Activities. Chapter 16 specifies how to develop External Agents. Appendix A lists all the reserved words in the Brahms agent language. Appendix B lists all the predefined Brahms facts and beliefs. Appendix C gives the complete grammar of the Brahms agent language.
