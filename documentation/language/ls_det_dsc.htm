Language Specification - Detectable (DET) - Description

Prev
Description Syntax Semantics
Next

A detectable is a declarative statement defining first which state changes an agent or object can detect and second what action results from detecting the state change.

Detecting facts is a two-step process. In the first step the agent or object detects the fact and the fact becomes a belief for the agent or object. No matter what the right hand side is in the form OA=V and OA=O the fact will become a belief for the agent or object. In the second step the beliefs of the agents or objects are matched with the detectable definition and if there is a positive match, the detectable action is executed.

Note that these two steps are independent for agents and objects, i.e. regardless of whether the fact is present in the world, the second step is performed. This means that if the agent or object has received a matching belief through a communication, the belief will trigger the action of the detectable.
