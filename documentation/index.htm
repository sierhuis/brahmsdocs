Documentation

Overview

The documentation page gives access to the latest version of the language specification, currently version 2.12, the tutorial, the API for the virtual machine, lists published papers, some downloadable some also viewable in HTML and will in the near future give access to tips and tricks on developing Brahms models. 

The language specification can be downloaded in HTML or Word format from the download page for local viewing in case you are not always able to connect to the internet.

A couple of new language features have been added to the language specification. Version 2.12 allows a modeler to: 

  • Define gesture activities. Gesture activities are only used to drive gesture visualizations in a virtual reality environment. Currently Brahms interfaces with OWorld/Adobe Atmosphere for visualizing Brahms simulations in a three-dimensional worlds.
  • Define create area activities. Create area activities are used to allow agents to create new areas at run-time, position these new areas in an area containment hierarchy and to position agents or objects within this new area.
  • Use the relational operators >, >=, <, <= in detectable conditions in addition to = and !=.

The virtual machine API currently contains API functions to develop Java Activities and External Agents.
