Tutorial Files

 

The following zip file contains folders with the Atm Tutorial files, at different stages of its evolution. You can use these files for comparison and for help. You can also use them to see immediately how a Brahms model work. The folder called ‘Final’ contains the complete version of the scenario. The others contain the scenario as modeled up till a certain section of this tutorial: Lesson VII, VIII, and IX.

 

Brahms_Tutorial_Files.zip

 

To use these files, you must copy the content of the folder whose scenario you want to run, and past it into the Atm model folder on your machine as discussed in the installation explanations. You can generate the xml files and then compile the scenario by:

 

C:\Program Files\Brahms\PersonalAgent\bc.bat c:\Brahms c:\brahms\projects\Atm\AtmModel.b

 

followed by

 

bvm projects.Atm.AtmModel

 

which will create the history file of the simulation in the database folder in PersonalAgent. You can recognize this file because it is a .txt file with the name of the file you compiled, with date and time

 

Thereafter, open the application AgentViewer , and click on the top left menu button, called ‘Parse New File’. Look for the history file when the selection window opens, and once you find it, select it. The AgentViewer will create a new MySQL database with the history of the simulation and then it will actually start parsing the file. If the parsing completes without sending out error messages, it will open the database and display the agents that were part of the simultion.
